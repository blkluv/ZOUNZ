# ZAOUNZ - Farcaster Music NFT Mini App\n\n**Status**: MVP ready for implementation (mint.club integration)\n**Timeline**: Deploy by EOD this week\n**Tech Stack**: React 19 + Vite + Express + mint.club API + Farcaster SDK\n\n## Quick Start\n\n### Prerequisites\n- Node.js 20+\n- npm/yarn\n- mint.club API key (get free at https://mint.club)\n\n### Setup\n\n1. **Clone & Install**\n   \`\`\`bash\n   git clone https://github.com/bettercallzaal/ZOUNZ.git\n   cd ZOUNZ\n   npm install\n   \`\`\`\n\n2. **Environment Setup**\n   \`\`\`bash\n   cp .env.example .env.local\n   # Edit .env.local and add:\n   # - MINT_CLUB_API_KEY (from https://mint.club)\n   # - APP_URL=http://localhost:5173 (for local dev)\n   \`\`\`\n\n3. **Run Local Dev**\n   \`\`\`bash\n   npm run dev:all\n   # Frontend: http://localhost:5173\n   # API: http://localhost:3001\n   \`\`\`\n\n## Features\n\n### Browse\n- 🎵 Browse 3 demo collections\n- ▶️ Play tracks inline\n- 📤 Queue for minting\n\n### Create\n- ➕ Create new collection\n- 📝 Add metadata\n- 🎵 Auto-associated with creator\n\n### Mint\n- ✨ Mint on Base network\n- ⚙️ Configure supply and royalty %\n- 🔗 View on Basescan\n- 📤 Cast to Farcaster\n\n## Deployment\n\n### Vercel (Recommended)\n1. Connect GitHub repo\n2. Set environment variables\n3. Deploy with `vercel deploy --prod`\n4. Register as Farcaster Mini App\n\n## Development Status\n\n### ✅ Completed\n- React 19 + Vite frontend\n- Browse, Create, Mint views\n- Express backend with 3 routes\n- Farcaster SDK integration\n- mint.club API structure\n\n### 🔨 Next Steps\n1.
